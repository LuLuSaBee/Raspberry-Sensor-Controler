## How to use
#### To use DHT11
```command line
sudo pip3 install dht11
```

#### To use PM2.5 Sensor (Adafruit 3686 PM2.5 Air Quality Sensor )
```command line
sudo pip3 install adafruit-circuitpython-pm25
```
